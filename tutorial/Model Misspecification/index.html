    return torch.cat((x.mean(axis=-1), x.std(axis=-1)**2), dim=1)

    Augment simulator with Gaussian noise model¶

    Standard simulator and prior: $$ x = f(\theta) \ \theta \sim p(\theta) $$

    Augmented simulator $$ x = f(\theta) + \xi(\gamma) \ \theta \sim p(\theta) \ \gamma \sim \Gamma(1, 2), $$ where \(\xi(\gamma) \sim \mathcal{N}(0, \gamma)\) is a Gaussian noise model with scale parameter \(\gamma\)

    Augmented simulator $$ x = f(\theta) + \gamma \ \theta \sim p(\theta) \ \gamma \sim \mathcal{N}(0, 1), $$

    def augmented_simulator(simulator, noise_model, theta, gamma):
    return simulator(theta) + noise_model(gamma)

    return gamma * torch.randn(gamma.shape)
